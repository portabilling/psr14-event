    public register(string[] $patterns, mixed[] $handlers): self
    Registre set of patterns to match and set of handlers to call if any pattern match

    Registre set of patterns to match and set of handlers to call if any pattern match

    While it is possible to setup multiple patterns and multiple handlers in one record, please be ecouraged to use 'one pattern to multiple handlers' or 'multiple patterns to one handler' approach.

    If your configuration will allow to call a handler meltiple times because it listed in two and more register() calls, and patterns will math - the same handler will be called multiple times.

    Parameters

    $patterns  

    Array of patterns to match agains event type. If any of array match the event type - all the handlers will run in order of given
    $handlers  

    Array of handlers to run. Basically, it should ne array of callable, any item which is not callable will try to resolve to callable by resolveHandler(), which do nothing for this base class. Override it if you wish to resolve your definition into callable.
